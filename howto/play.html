What Are ROM Patches?

A ROM patch is a file that contains changes made to a game, like new levels or mechanics. Since sharing the full game is illegal, we only share the modifications as patch files (usually .ips, .bps, or .ups).

These patches are applied to a clean (unmodified) ROM to produce a modified version of the game.

Why Use Patches?

Legal reasons: Distributing full ROMs violates copyright laws. Patches only contain original content created by the modder.

How to Patch a ROM

If your patch is available on this site, you can use the built in patcher on the site.

  1. Get a clean ROM: This must be an unmodified version of the original game.
  2. Open the hack page: Each hack that is hosted on the site has its own dedicated page.
  3. Go down to the online patcher: Below the images you can find a section for the online patcher.
  4. Select the ROM file
  5. Click "Apply patch". The tool will generate the patched ROM.
  6. Download the patched file and play it using your favorite emulator.

If your patch is not from this site, you can go to the externally hosted version of RomPatcherJS and then follow the tutorial below.
You can use RomPatcherJS to apply patches directly in your browser.

  1. Get a clean ROM: This must be an unmodified version of the original game.
  2. Download a patch: For example a .ips, .bps, or .ups file.
  3. Open RomPatcherJS.
  4. Select the ROM file and the patch file in the tool.
  5. Click "Apply patch". The tool will generate the patched ROM.
  6. Download the patched file and play it using your favorite emulator.

  • Always keep a backup of your original ROM.
